# Postavitev slik na predlogo Seat spletne strani

**Ta projekt je bil narejen za vajo z uporabo Canvasa in SVG slik.**

## Opis projekta
Projekt prikazuje preprosto predlogo domače spletne strani znamke SEAT, v katero so umeščene slike (JPEG/AVIF) in SVG element. Namen vaje je bil vaditi postavitev slik, delo s uporabo Canvasa za risanje logota

## Funkcionalnosti
- Enostavna hero sekcija s sliko avtomobila in opisom

- Osnovne CSS postavitve in stilizacija

- Minimalen JavaScript za gladko pomikanje po strani

## Struktura projekta
1. `seat.html` — glavna HTML datoteka

2. `css/style.css` — slogovne datoteke

3. `js/script.js` — skripte
  - `js/seat_logo.js` — skripta za risanje seat logota

4. `img/` — uporabljene slike


## Kako zagnati
1. Repozitorij preneseš.

2. Klikneš na link v repositoriju (v zgornjem desnem kotu).

3. Odpreš eno od HTML datotek v brskalniku:

Ni potrebna dodatna namestitev – dovolj je sodoben spletni brskalnik.


## Opombe
- AVIF slike so bile že prisotne v spletnem mestu in sem jih ohranil; ob tem sem v Canvasu pripravil dodatni logotip kot SVG (z možnostjo spreminjanja barv) in ga dodal v projekt.

- Najprej sem poiskal originalno SVG datoteko logotipa SEAT, nato sem v Canvasu narisal lastno različico logotipa in sliko vključil na spletno stran.

- Originalni SVG sem izvozil tudi kot PNG in ga umestil na dno spletne strani.
